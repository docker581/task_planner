# task_planner

## Описание проекта
Проcтой планировщик задач на Flask

## Стек технологий
- Python 3.8.0
- Flask 2.2.3
- Flask-SQLAlchemy 3.0.3
- Flask-WTF 1.1.1
- Bootstrap 4.5.3
- SQLite 3.28.0

## Команды
### Клонирование репозитория
```bash
git clone https://github.com/docker581/task_planner
```

### Установка зависимостей
```bash
pip install -r requirements.txt
```

### Запуск приложения
```bash
python run.py
```
